HTML FORMATTING TAGS

TAGS NAME FORMAT FUNCTIONS
Bold Text <b> </b> The HTML bold element defines bold text, without any extra importance.
Important Text <Strong> </Strong> The HTML <Strong> element defines strong text, with added semantic "strong" importance.
italic Text <i> </i> The HTML <i> element element defines italic text, without any extra importance.
Emphasized Text <em> </em> The HTML <em> element defines emphasized text, with added semantic importance.
Marked/Highlighted Text <mark> </mark> The HTML <mark> element defines marked or highlighted text.
Deleted Text <del> </del> The HTML <del> element defines deleted (removed) text.
Inserted Text <ins> </ins> The HTML <ins> element defines inserted (added) text.
Subscripted Text <sub> </sub> The HTML <sub> element defines subscripted text.
Superscripted Text <sup> </sup> The HTML <sup> element defines superscripted text.
